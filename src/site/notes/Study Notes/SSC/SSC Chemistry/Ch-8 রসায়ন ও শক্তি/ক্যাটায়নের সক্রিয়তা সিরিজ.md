# ক্যাটায়নের সক্রিয়তা সিরিজ

### **ক্যাটায়নের সক্রিয়তা সিরিজ (The Reactivity Table)**

রসায়নের জগতে কিছু ধাতু হলো খুব বেশি বিক্রিয়া করে (বেশি সক্রিয়)। আবার কিছু ধাতু একদম শান্তশিষ্ট, সহজে কারো সাথে বিক্রিয়া করতে চায় না (কম সক্রিয়)। এই সক্রিয়তার সিরিজটা আমাদের বলে দেয়, কোন ধাতু কতটা শক্তিশালী।

### **মনে রাখার ছন্দ:**

>[!Tip] লিজা কে না কে ম্যাগাইভার এলো যেন ফিরে সোনা পাবে হায় কোর্টে হাজীর আজ পিটাবে আমায়।
> 

এবার চলো, এই ছন্দটাকে ভেঙে টেবিলে সাজিয়ে ফেলি:

| ছন্দের অংশ | মৌল (Element) | প্রতীক | সক্রিয়তা |
| --- | --- | --- | --- |
| **লি**জা | লিথিয়াম (Lithium) | $Li^+$ | **↑** সবচেয়ে বেশি সক্রিয় |
| **কে** | পটাশিয়াম (Potassium) | $K^+$ |  |
| **না** | সোডিয়াম (Sodium) | $Na^+$ |  |
| **কে** (ক্যালসিয়াম) | ক্যালসিয়াম (Calcium) | $Ca^{2+}$ |  |
| **ম্যা**গাইভার | ম্যাগনেসিয়াম (Magnesium) | $Mg^{2+}$ |  |
| **এ**লো | অ্যালুমিনিয়াম (Aluminium) | $Al^{3+}$ |  |
| **যে**ন | জিঙ্ক (Zinc) | $Zn^{2+}$ |  |
| **ফি**রে | আয়রন (Iron) | $Fe^{2+}$ |  |
| সোনা | টিন (Tin) | $Sn^{2+}$ |  |
| **পা**বে | লেড (Lead) | $Pb^{2+}$ |  |
| হায় | **হাইড্রোজেন (Hydrogen)** | **$H^+$** | **(এটা হলো Standard)** |
| **কো**র্টে | কপার (Copper) | $Cu^{2+}$ |  |
| **হা**জীর | মার্কারি (Mercury) | $Hg^{2+}$ |  |
| **আজ** | সিলভার (Silver) | $Ag^+$ |  |
| **পি**টাবে | প্লাটিনাম (Platinum) | $Pt^{2+}$ |  |
| **আ**মায় | গোল্ড (Gold) | $Au^{3+}$ | **↓** সবচেয়ে কম সক্রিয় |

### **টিপস: এই টেবিলটা কীভাবে ব্যবহার করবে?**

1. **যে যত উপরে, সে তত বস (বেশি সক্রিয়):** টেবিলের যত উপরের দিকে যাবে, ধাতুগুলো তত বেশি বিক্রিয়া করতে চায় এবং তত বেশি শক্তিশালী। লিথিয়াম, পটাশিয়াম হলো এই গ্রুপের "বিগ বস"!
2. **হাইড্রোজেন হলো সীমানা:** হাইড্রোজেনকে এখানে একটা বর্ডার লাইনের মতো ধরা হয়। এর উপরে যারা আছে, তারা অ্যাসিড থেকে হাইড্রোজেনকে সরিয়ে দিতে পারে। নিচেরগুলো পারে না।
3. **তড়িৎ বিশ্লেষণে কে জিতবে?:** এটাই সবচেয়ে গুরুত্বপূর্ণ!
    - ক্যাথোডে (-) চার্জমুক্ত হওয়ার প্রতিযোগিতায়, এই টেবিলের **যে নিচে থাকবে, সে জিতবে**।
    - **উদাহরণ:** $Na^+$ আর $H^+$ এর মধ্যে রেস লাগলে, $H^+$ যেহেতু $Na^+$ এর অনেক নিচে আছে, তাই $H^+$ জিতে গিয়ে হাইড্রোজেন গ্যাস তৈরি করবে।
